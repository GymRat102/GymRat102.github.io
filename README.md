## About Theme

This blog uses the [jekyllBear theme](https://github.com/knhash/jekyllBear), but due to GitHub Pages only supporting [a limited set of themes](https://pages.github.com/themes/), I have manually copied the theme files into this repository and set `theme: null` in the configuration.

The theme is originally developed by [knhash](https://github.com/knhash) and is licensed under the MIT License. All copyright and credit for the original theme belong to the original author. At this stage, no modifications have been made to the theme.
